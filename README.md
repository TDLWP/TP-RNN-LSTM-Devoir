# TP : Réseaux de Neurones Récurrents (RNN & LSTM)

Ce dépôt contient la solution complète du TP sur les RNN, LSTM et BPTT, structuré sous forme de Notebook Jupyter pour une meilleure lisibilité et interactivité.

## Contenu
- `TP_RNN_LSTM.ipynb` : Travail complet incluant code Python et justifications théoriques.
- `requirements.txt` : Liste des dépendances nécessaires.

## Installation et Exécution
1. Clonez ce dépôt.
2. Installez les dépendances :
   ```bash
   pip install -r requirements.txt
   ```
3. Lancez le Notebook :
   ```bash
   jupyter notebook TP_RNN_LSTM.ipynb
   ```
